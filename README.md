# 🎓 Student Management System (Java)

## 📌 Project Description
This is a console-based Java application built using Object-Oriented Programming (OOP) concepts.  
It allows users to manage student records with basic operations like adding, displaying, searching, and deleting students.

---

## 🚀 Features
- Add new student
- Display all students
- Search student by ID
- Delete student by ID
- Menu-driven program using loop

---

## 🛠️ Technologies Used
- Java
- OOP (Encapsulation, Classes, Objects)
- ArrayList (Dynamic data storage)

---

## 🧠 Concepts Used
- Classes and Objects
- Constructor
- Encapsulation
- ArrayList
- Loop (while loop)
- Switch-case
- Lambda expression (removeIf)
- Iterator (alternative for deletion)

---

## ▶️ How to Run the Project

1. Open Eclipse / any Java IDE  
2. Create a Java Project  
3. Add the files:
   - `Student.java`
   - `Main.java`  
4. Run `Main.java`
